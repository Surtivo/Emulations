# CHIP-8 Emulator / Interpreter

## Instructions:

1 - Go to the folder "CHIP-8" and compile the program using the command:
      
      $make 

2 - Run the program using:

      $./chip8 Roms/~ROM_NAME~

## Programs Used:
Project developed in Visual Studio Code and tested with GCC on a Lubuntu Virtual Machine.
Used libsdl12 for graphical interface.

### Observation:
